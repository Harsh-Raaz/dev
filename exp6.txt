✅ DevOps Lab – Experiment 6
🎯 Aim

To install Jenkins, integrate Maven, and build a Java application using a Jenkins Freestyle Job.

🔷 Procedure
🔸 Step 1: Launch EC2 Instance
OS: Ubuntu
Start instance
Connect using SSH
ssh -i your-key.pem ubuntu@your-public-ip
🔷 INSTALLATION
🔸 Step 2: Update System & Install Java (JDK 21)
sudo apt update
sudo apt install openjdk-21-jdk -y
✔ Verify:
java -version
🔸 Step 3: Install Maven (3.9.6)
wget https://archive.apache.org/dist/maven/maven-3/3.9.6/binaries/apache-maven-3.9.6-bin.tar.gz

sudo tar -xzf apache-maven-3.9.6-bin.tar.gz -C /opt/

echo 'export PATH=/opt/apache-maven-3.9.6/bin:$PATH' | sudo tee /etc/profile.d/maven.sh

source /etc/profile.d/maven.sh
✔ Verify:
mvn -version
🔸 Step 4: Install Jenkins
curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key | sudo tee \
/usr/share/keyrings/jenkins-keyring.asc > /dev/null

echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
/etc/apt/sources.list.d/jenkins.list > /dev/null

sudo apt update
sudo apt install jenkins -y
🔸 Step 5: Start Jenkins
sudo service jenkins start
sudo service jenkins status
🔸 Step 6: Open Port 8080 (AWS)
EC2 → Security Group → Inbound Rules
Add:
Port: 8080
Source: 0.0.0.0/0
🔸 Step 7: Access Jenkins
http://your-public-ip:8080
🔸 Step 8: Unlock Jenkins
sudo cat /var/lib/jenkins/secrets/initialAdminPassword

👉 Paste password in browser
👉 Install suggested plugins
👉 Create user

🔷 JENKINS CONFIGURATION
🔸 Step 9: Install Maven Plugin
Go to Manage Jenkins → Plugins
Install:
✔ Maven Integration Plugin
🔸 Step 10: Configure Maven in Jenkins

Go to:

Manage Jenkins → Tools → Maven
Field	Value
Name	Maven 3.9.6
Install automatically	❌ Uncheck
MAVEN_HOME	/opt/apache-maven-3.9.6
🔷 CREATE JENKINS JOB
🔸 Step 11: Create Freestyle Job
New Item
Select Freestyle Project
Name it → OK
🔸 Step 12: Configure Job
✔ Source Code Management
https://github.com/mvsanthi123/simple-java-maven-app.git
✔ Build Step
Add Build Step → Invoke top-level Maven targets
Field	Value
Maven Version	Maven 3.9.6
Goals	clean install
🔷 BUILD & EXECUTE
🔸 Step 13: Run Build
Click Build Now
Check Console Output
🔸 Step 14: Check JAR File
sudo su - jenkins
cd /var/lib/jenkins/.m2/repository/com/mycompany/app/my-app/1.0-SNAPSHOT/
ls

👉 Output:

my-app-1.0-SNAPSHOT.jar
🔸 Step 15: Run Application
java -jar my-app-1.0-SNAPSHOT.jar

👉 Output:

Hello World
✅ Result
Jenkins installed successfully
Maven integrated with Jenkins
Java project built using Jenkins
.jar file executed successfully
